Custom Internal-Only Admonition¶

We've added a custom admonition type for internal-only content:

Internal-Only Content

This content only appears in the internal build of the documentation. It will be automatically hidden in the external build.

Collapsible Internal Admonition¶

You can also create collapsible versions:

Click to reveal internal details

This is a collapsible internal-only section.

It contains implementation details or other information that should only be visible to internal users.
