## Quick Start

Get up and running quickly with SiteTracker's platform:

=== "macOS"
    ```bash
    brew install sitetracker-cli
    sitetracker init --api-key YOUR_API_KEY
    ```

=== "Windows"
    ```powershell
    choco install sitetracker-cli
    sitetracker init --api-key YOUR_API_KEY
    ```

=== "Linux"
    ```bash
    curl -sSL https://get.sitetracker.com | bash
    sitetracker init --api-key YOUR_API_KEY
    ```
